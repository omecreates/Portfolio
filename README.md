## Installation

Clone the repository:

```bash
git clone https://github.com/omecreates/your-portfolio-repo.git
```

Navigate into the project:

```bash
cd your-portfolio-repo
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
